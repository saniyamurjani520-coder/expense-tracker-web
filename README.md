# Expense Tracker Web App  A web-based expense tracker built using Flask and SQLite.  ## Features - Add expenses - View all expenses - Delete expenses - Category and date tracking  ## Tech Stack - Python - Flask - SQLite  ## How to Run 1. Install Flask: pip install flask 2. Run: python app.py 3. Open browser: http://127.0.0.1:5000
